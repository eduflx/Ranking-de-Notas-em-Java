# Ranking-de-Notas-em-Java
Programa em Java que reliza calculos sobre notas em um vetor.
O Programa é baseado em 3 funções dentro da classe: 

* Calcular a média: Calcula a media dos valores do vetor.
* Acima da média: Calcula as notas que foram acima da média.
* Maior nota: Retorna a maior nota dentre as outra do vetor.

O programa retorna os resultados dentro da classe principal, que chama as funções e as executa.

Exercicio 1 da Lista de exercícios nº3 de Extrutura de Dados 1 do curso de ADS do IFTM Campus Patrocínio.
